Pendidikan

Awal pendidikan saya, saya bersekolah di SDIT Yapidh dan melanjutkan juga ke SMPIT Yapidh, jadi saya bersekolah di Yapidh selama 9 tahun. Lalu saat SMA saya ingin mencoba bersekolah di sekolahan negeri dan juga katanya agar mempermudah masuk ke perguruan tinggi negeri. Sehingga saya mendaftar ke sekolah SMAN 11 Bekasi dan alhamdulillah nya saya diterima melalui jalur NEM. Pada saat kelas 12 saya bingung mau melanjutkan ke perguruan tinggi mana dan pada saat itu saya juga mengalami masa yang sangat bimbaaang sekali antara ingin linjur atau tidak. Setelah saya merenung akhirnya saya memutuskan untuk linjur dan saya pun belajar ketertinggalan saya untuk mengejar materi UTBK soshum. Dan saya pun memilih untuk melanjutkan pendidikan saya ke UIN Jakarta. Saya juga mengikuti ujian mandirinya, dan sayangnya saya tidak diterima, tetapi alhamdulillahnya saya diterima melalui jalur utbknya. Sehingga sekarang saya berada di UIN Jakarta, jurusan Ilmu Perpustakaan, Fakultas Adab dan Humaniora dan sekarang saya sudah berada di semester 5.
